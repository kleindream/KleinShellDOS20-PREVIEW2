# Klein Shell DOS 2.0

Klein Shell DOS 2.0 is a retro-inspired command-line adventure game that simulates a mysterious DOS-like computer system.

Explore directories, investigate hidden files, recover lost records, unlock achievements, complete missions, and uncover the story behind a forgotten machine.

Inspired by classic DOS environments and vintage computing, the game combines command-line interaction, digital archaeology, exploration, and mystery into a unique experience.

## Features

* DOS-style interface
* Multiple color themes
* Mission system
* Achievement system
* Hidden files and secrets
* Investigation and exploration gameplay
* Story-driven progression
* Retro computing atmosphere
* Open source (MIT License)
